💡 Smart Home Automation Dashboard
IoT ke through ghar ke devices ko remotely control karo. Har device ek GPIO pin se connected hoti hai — turn ON/OFF karo aur energy consumption monitor karo.

📖 THEORY

Smart Home Automation mein IoT devices (bulbs, fans, AC, locks) ko internet ke through control kiya jaata hai. Raspberry Pi ya ESP8266 ka use karke GPIO pins se devices ko on/off kiya jaata hai.

  • GPIO (General Purpose Input/Output) pins ke through relay control
  • MQTT ya HTTP API se commands send hoti hain
  • Scheduling: time-based automation (e.g., lights off at 11pm)
  • Energy monitoring: wattage track karke savings calculate karna
