📡 MQTT Protocol Simulator
MQTT (Message Queuing Telemetry Transport) IoT ka backbone protocol hai. Ye lightweight publish-subscribe model use karta hai. Is simulator mein Publisher aur Subscriber bano aur Broker ke through messages dekho.

📖 THEORY

MQTT ek lightweight messaging protocol hai jo IoT devices ke beech communication ke liye use hota hai. 3 main components hain:

  • Publisher: Data bhejne waala device (e.g., temperature sensor)
  • Broker: Message routing server (e.g., Mosquitto, HiveMQ)
  • Subscriber: Data receive karne waala device (e.g., dashboard)
  • Topic: Message ka address (e.g., home/bedroom/temp)
  • QoS (Quality of Service): 0=At most once, 1=At least once, 2=Exactly once
