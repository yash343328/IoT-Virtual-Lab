🌡️ Temperature & Humidity Sensor
DHT11/DHT22 sensor simulation. Sensor se data read karo, threshold set karo, aur alerts observe karo. IoT mein sensor monitoring ka pehla step yahi hai.

📖 THEORY

DHT11/DHT22 sensors temperature aur humidity measure karte hain. IoT systems mein ye data microcontroller (Arduino/Raspberry Pi) ke through cloud ya dashboard pe bheja jaata hai.

  • DHT11: 0–50°C temp, 20–80% humidity (low cost)
  • DHT22: -40–80°C temp, 0–100% humidity (high accuracy)
  • Data protocol: Single-wire serial communication
  • Threshold alerts: Define karo kab "normal" aur kab "danger"
